Jak pracovat s tímto listem

Tento list je pracovní hra. Neslouží jen ke klikání, ale k pochopení, jak spolu souvisí složky, instalace, backend a šablony.

1. Co je cílem tohoto listu

Cílem je složit vlastní backend po vrstvách. Nejprve se skládá struktura projektu, potom backend vracející JSON a nakonec backend, který načítá HTML šablonu a static files. Výstupem nemá být jen „správně naklikaná hra“, ale pochopení toho, kam patří .venv, kde leží main.py, co vrací route a jak FastAPI obsluhuje stránku.

2. Jak se tento list ovládá

  1. V levém panelu vyber misi.
  2. Přečti si zadání mise.
  3. Kliknutím na uzel ho přidej na plochu.
  4. Uzel můžeš myší přesouvat.
  5. Klikni na pravý výstupní bod jednoho uzlu a potom na levý vstupní bod druhého uzlu.
  6. Takto vytvoříš vztah mezi dvěma částmi backendu.
  7. Průběžně si čti vysvětlení v pravém panelu.
  8. Až budeš mít hotovo, klikni na Zkontrolovat řešení.

3. Jak poznat, že je řešení správně

  • Na pracovní ploše musí být správné uzly.
  • Uzly musí být propojené ve správných vztazích.
  • Pokud je řešení správně, stavový box zezelená a v pravém panelu se objeví výsledný výstup.
  • Pokud řešení správně není, stavový box napoví, co je špatně.

4. Aktuální zadání

Vyber misi a zde se zobrazí její zadání.

5. Co je výstupem jednotlivých misí

  • Mise 1: vygeneruje strom složek a souborů projektu + instalační příkazy.
  • Mise 2: vygeneruje main.py pro JSON endpoint nad SQLite databází.
  • Mise 3: vygeneruje main.py pro HTML šablonu a static files + ukázku index.html.

Slovník pojmů

Základní pojmy, které se v tomto listu objevují opakovaně.

Backend

Část aplikace, která běží na serveru. Zpracovává logiku, komunikuje s databází a vrací odpovědi.

Route

Adresa backendu, například / nebo /reservations. Každá route obvykle spouští určitou Python funkci.

JSON

Textový formát pro předávání dat. Ve FastAPI je to výchozí forma odpovědi, když vrátíš Python slovník nebo seznam.

Template

HTML šablona, do které backend může posílat data. Ve FastAPI se často používá Jinja2.

Static files

Statické soubory jako CSS, JavaScript nebo obrázky. Ve FastAPI se připojují přes app.mount("/static", ...).

.venv

Virtuální prostředí projektu. Drží Python balíčky odděleně od ostatních projektů.

Uvicorn

ASGI server, který FastAPI aplikaci skutečně rozběhne a zpřístupní ji na adrese a portu.

SQLite

Souborová databáze. V našem projektu je uložená například v souboru reservations.db.
